How to navigate through git bash during command lines.

It is a tool that allows you to directly inform the system how to behave, instead of using the graphic interface.
This not only speeds up the navigation process, it also allows more advance functions that are not apparent using the graphic interface.
Common command lines:

  • cd (change directory)
    Commonly used to change locations, and navigate through files.
  • ls (list directory contents)
    Used to understand what is exisiting inside the directory/folder.
  • mkdir
    To create a new directory
  • rmdir
    removing a directory
  • mv
    moving a file or directory
