display: inline

The display property specifies the display behavior of an element.
In the case of display:inline, it displays an element (e.g. span, a, img) in the same line as other elements. However, any height and width properties will not take effect. Also, with display: inline both the top and bottom margins/paddings are not respective.

display: inline-block

Both display: inline and display: inline-block allow the element to be placed side by side (same line). However, unlike inline, inline-block allows you to set a width and height property on the element. Also, with inline-block, both the top and bottom margins/paddings are respected.

Example

Exercitation ad qui sunt ipsum laboris mollit reprehenderit.Velit ullamco mollit nostrud enim eu proident mollit aliquip eiusmod sint eu.
Cupidatat adipisicing Ut magna ipsum ea consectetur aliqua duis occaecat laboris non anim quis consectetur esse quis.

display: block

Compared, inline-block, block would add a line-break after the element, however, both margins/padding will remain respective. Similar to inline-block, block allows you to set a height and width property to the element.
